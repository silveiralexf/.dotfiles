# dotfiles

Personal dotfiles... buyers beware!

```sh
ln -s ~/.dotfiles/nvim ~/.config/nvim
ln -s ~/.dotfiles/config/tmux.conf ~/.tmux.conf
ln -s ~/.dotfiles/config/kitty.conf ~/.config/kitty/kitty.conf
ln -s ~/.dotfiles/config/alacritty.yml ~/.config/alacritty/alacritty.yml
ln -s ~/.dotfiles/scripts/pass-user.sh ~/pass-user.sh
ln -s ~/.dotfiles/config/pet-config.toml ~/.config/pet/config.toml
ln -s ~/.dotfiles/config/zshrc ~/.zshrc
```
