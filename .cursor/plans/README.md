# Plans

Implementation plans live here. Naming: **`YYYY-MM-DD_<username>_<slug>.plan.md`**.

Example: `2025-02-27_felipe.silveira_wezterm-keybinding.plan.md`

Plans are created by @planner and referenced by agents during execution and review.

**Renaming:** Non-conforming plans are renamed on pre-commit by `scripts/rename-cursor-plans.sh`. Run `task plan:rename -- --dry-run` to preview, or `task plan:rename` to fix now. See `.cursor/rules/dotfiles-plan-naming.mdc`.
