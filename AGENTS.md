# Agentic workflow for dotfiles

For AI-assisted changes, use the **dotfiles agentic workflow**. Start with **@planner** and describe your task.

- **Workflow:** `.cursor/AGENTIC_WORKFLOW.md`
- **Triggers / how to invoke:** `.cursor/WORKFLOW_TRIGGERS.md`
- **Gates:** `.cursor/GATE_SYSTEM.md`

Flow: plan (bite-sized tasks) → implement (config or taskfile) → @code-reviewer → finish branch (task precommit, then merge/PR/keep/discard).
